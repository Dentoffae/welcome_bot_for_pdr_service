# Welcome Bot — Приветственный бот для автосервиса (PDR)

Бот на базе OpenAI, который отвечает клиентам от имени мобильного автосервиса по удалению вмятин без покраски (Paintless Dent Repair). Поддерживает русский, английский и арабский язык. Доступны два режима запуска: **веб-интерфейс** (Telegram-стиль) и **консоль**.

---

## Структура проекта

```
WELCOME_BOT/
├── web.py             # Веб-интерфейс (Telegram-стиль) — открывается в браузере
├── bot.py             # Консольная версия — диалог прямо в терминале
├── templates/
│   └── index.html     # UI веб-интерфейса
├── prompts.json       # Заготовленные промпты (роль, контекст, сценарий бота)
├── .env               # Ваши API-ключи (НЕ публиковать!)
├── .env.example       # Шаблон .env — скопируйте и заполните
├── requirements.txt   # Зависимости Python
└── venv/              # Виртуальное окружение (создаётся при установке)
```

---

## Шаг 1 — Установите Python

Убедитесь, что у вас установлен **Python 3.10 или новее**.

Проверьте версию в терминале:
```
python --version
```

Если Python не установлен — скачайте с [python.org](https://www.python.org/downloads/) и установите, **поставив галочку «Add Python to PATH»**.

---

## Шаг 2 — Создайте виртуальное окружение

Откройте терминал в папке `WELCOME_BOT` и выполните:

```
python -m venv venv
```

Активируйте окружение:

- **Windows (PowerShell):**
  ```
  venv\Scripts\Activate.ps1
  ```
- **Windows (CMD):**
  ```
  venv\Scripts\activate.bat
  ```
- **macOS / Linux:**
  ```
  source venv/bin/activate
  ```

После активации в начале строки терминала появится `(venv)`.

---

## Шаг 3 — Установите зависимости

```
pip install -r requirements.txt
```

Это установит:
- `openai` — официальная библиотека OpenAI
- `requests` — для HTTP-запросов
- `python-dotenv` — для чтения переменных из `.env`

---

## Шаг 4 — Настройте API-ключ

1. Скопируйте файл `.env.example` и переименуйте копию в `.env`:
   ```
   copy .env.example .env
   ```

2. Откройте `.env` и вставьте ваш ключ OpenAI:
   ```
   OPENAI_API_KEY=sk-proj-ВАШ_КЛЮЧ_ЗДЕСЬ
   ```

3. Если хотите использовать другую модель (по умолчанию `gpt-3.5-turbo`), добавьте строку:
   ```
   OPENAI_MODEL=gpt-4o
   ```

> Получить ключ OpenAI можно на [platform.openai.com/api-keys](https://platform.openai.com/api-keys)

---

## Шаг 5 — Запустите бота

### Вариант A: Веб-интерфейс (рекомендуется)

```
python web.py
```

Затем откройте в браузере: **http://localhost:5000**

Вы увидите интерфейс, похожий на Telegram:
- Список промптов в левой колонке
- Чат с пузырьками сообщений справа
- Кнопка тест-запроса внизу
- Счётчик токенов в реальном времени

### Вариант B: Консоль

```
python bot.py
```

## Как работает диалог

После запуска скрипт задаст несколько вопросов:

1. **Выберите номер запроса** — нажмите `1` чтобы использовать готовый промпт «Приветственный бот для автосервиса», или `0` для ввода своего.

2. **Тестовый ввод** — нажмите `Enter` чтобы использовать готовое тестовое сообщение (`"Привет! я хочу узнать стоимость ваших услуг."`), или введите своё.

3. **Параметры генерации** — нажмите `Enter` везде чтобы принять значения по умолчанию:
   - Температура: `0.7` (чем выше — тем более «творческие» ответы)
   - max_completion_tokens: `1024`
   - frequency_penalty: `0.5`

4. Далее идёт обычный **диалог** — вы пишете как клиент, бот отвечает как менеджер автосервиса.

## Настройка промптов

Промпты хранятся в файле `prompts.json`. Каждый промпт содержит:

| Поле         | Описание |
|--------------|----------|
| `id`         | Номер для выбора в меню |
| `name`       | Название промпта |
| `role`       | Роль бота (кто он?) |
| `context`    | Контекст бизнеса |
| `question`   | Задача, которую решает бот |
| `format`     | Правила поведения и стиль ответов |
| `test_input` | Тестовое сообщение для быстрого запуска |

Вы можете добавить новые промпты, скопировав существующий блок и изменив поля.

## Часто встречающиеся ошибки

**`OPENAI_API_KEY не найдена`**
→ Проверьте, что файл `.env` существует и содержит правильный ключ.

**`openai не установлена`**
→ Убедитесь, что виртуальное окружение активировано (`(venv)` в терминале) и выполните `pip install -r requirements.txt`.

**`Error 401 / Incorrect API key`**
→ Ключ недействителен или устарел. Создайте новый на [platform.openai.com](https://platform.openai.com/api-keys).

**`Error 429 / Rate limit`**
→ Превышен лимит запросов. Подождите минуту или пополните баланс аккаунта OpenAI.

## Быстрый старт (все шаги разом)

```
# Активировать окружение
venv\Scripts\Activate.ps1

# Установить зависимости (только при первом запуске)
pip install -r requirements.txt

# Запустить веб-интерфейс → открыть http://localhost:5000
python web.py

# ИЛИ запустить консольную версию
python bot.py
```

Сменить порт веб-сервера можно в `.env`:
```
WEB_PORT=8080
```
